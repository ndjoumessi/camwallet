# CamWallet

Application pour la gestion des transactions au Cameroun.

CamWallet est une solution de gestion de portefeuille et de transactions financières
adaptée au contexte camerounais (Mobile Money, transferts, suivi des dépenses).

## Contenu du dépôt

- `camwallet-mvp.html` — Prototype MVP de l'application (interface utilisateur).
- `camwallet_admin_dashboard.html` — Tableau de bord administrateur.
- `CamWallet_CDC_v1.0_llm.json` — Cahier des charges (spécifications du projet).

## Démarrage rapide

Ouvrez les fichiers `.html` directement dans un navigateur pour visualiser
les prototypes. Aucune installation requise pour le moment.
